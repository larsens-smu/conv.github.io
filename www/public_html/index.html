About Me

CIFITW stands for Convolution Image Filtering In the Web
Javascipt is a slow language, but it is very helpful to build and deploy applications in the web.

While it is typically easy to use a language like C++ to quickly apply a fitler to an image in JS it can be slower

This project leverages Emscripten to transpiles a fast convoltional library into JS for use in the browser

There is also a downloadable windows and mac version, leveraging Electron
